# qiita-sample

Qiita 記事で扱ったサンプルコード置き場です。  
C#、.NET、WinForms を中心に、動きを手元で確認しやすい小さめのサンプルをまとめています。

## このリポジトリの見方

- サンプルは記事ごとにフォルダを分けています
- 最初にこの README で対象フォルダを見つけ、その後で各フォルダ配下の `README.md` を見る流れが分かりやすいです
- 詳しい使い方や確認ポイントは各サンプル側に書いています

## サンプル一覧

| フォルダ名 | 内容 | 主な技術 | 補足 |
|---|---|---|---|
| `K35_LinqSample` | LINQ の動きや書き方を確認するサンプル | C# / LINQ / WinForms | 条件ごとの絞り込みや結果確認向け |
| `W01_FileSystemObserver` | ファイル変更監視の基本を確認するサンプル | C# / .NET Framework / WinForms / FileSystemWatcher | 監視開始・停止、イベント確認向け |

## 見る順

1. Qiita の記事で概要を確認  
2. この README から対象サンプルを選ぶ  
3. 対象フォルダ配下の `README.md` で使い方と見どころを確認  
4. その後でソースコードを見る  

## 動作環境

- Windows
- Visual Studio
- サンプルごとに対象フレームワークや前提が異なります
- 詳細は各フォルダ配下の `README.md` を見る形です

## 注意

- 学習用・検証用のサンプルを含みます
- 実運用へ入れる前には例外処理、ログ、再試行、権限、運用条件の確認が別途要ります
- 画面サンプルは理解しやすさを優先しています
