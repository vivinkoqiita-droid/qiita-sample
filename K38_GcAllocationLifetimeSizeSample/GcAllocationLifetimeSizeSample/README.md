# GcAllocationLifetimeSizeSample

WinForms で「割り当て」「長く残る」「サイズ」をライブ観測する学習用サンプルです。

## 見る順

1. シナリオを選ぶ
2. 開始を押して 5〜10 秒ほど流す
3. 保持クリア / GC強制実行 を押して変化を見る
4. 右側の説明で、どこを疑うべきかを読む

## 画面の読み方

- 作られる量: このサンプルが意図的に作った件数の増え方
- 長く残る量: 保持中件数と戻りにくさ
- 大きい確保: 85000 bytes 以上の確保回数
- 短いGC: Gen0 GC Count の増え方
- 重いGC: Gen2 GC Count の増え方
- 回収完了: Finalizer まで到達した件数

## シナリオ

- 文字列連結で細かく作る
- 小さいオブジェクトを毎回確保
- 長く残るように保持し続ける
- 大きい配列を毎回確保
- 大きい配列を保持し続ける

## 補足

この画面は `dotnet-counters` や Profiler の代替ではなく、記事の「最初に見る方向」を体感で理解するためのサンプルです。
