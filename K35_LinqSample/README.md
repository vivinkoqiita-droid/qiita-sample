# K35 LINQ 実行サンプル（WinForms）

Qiita 記事（K35）で扱う LINQ の挙動を、画面上のボタンで再現するサンプルです。  
「データ」「コード」「出力」「実行時間」を同じ画面で確認できるようにしています。

---

## 画面で見えるもの

- **データ表（左）**  
  Departments / Employees / Orders の固定データを表示します。

- **コード表示（右上）**  
  選択中デモのコード（抜粋）を表示します。  
  ※表示上は `Console.WriteLine(...)` ですが、実行時の出力先は画面右下の Console です。

- **Console（右下）**  
  実行ログ・結果を 1 行ずつ表示します。

- **Command / Elapsed（Console上部）**  
  直前に実行したデモ名と、実行時間（ms）を表示します。

---

## ボタンで得られること

### 共通

- **各デモボタン**：押すとデモを実行します  
  マウスオーバー中は、そのデモのコードをプレビュー表示します。
- **Clear**：Console の表示と、Command / Elapsed をクリアします

---

### 遅延実行

| ボタン | 得られること |
|---|---|
| 遅延① 列挙で動く | クエリは作っただけでは動かず、列挙した瞬間に評価されることを確認できます |
| 遅延② 2回列挙 | 同じクエリを2回列挙すると、同じ処理が2回走ることを確認できます |
| 遅延③ ToListで止める | `ToList()` で一度だけ評価して結果を固定できることを確認できます |

### 絞り込み

| ボタン | 得られること |
|---|---|
| Where | 条件で絞り込む基本形を確認できます |
| Take | 先頭から N 件だけ取る動きを確認できます |
| Skip | 先頭から N 件を飛ばす動きを確認できます |
| TakeWhile | 条件が成り立つ間だけ取り続ける動きを確認できます |
| SkipWhile | 条件が成り立つ間だけ飛ばし続ける動きを確認できます |

### 変換

| ボタン | 得られること |
|---|---|
| Select | 要素を別の形へ写す動きを確認できます |
| SelectMany | ネストをフラット化する動きを確認できます |

### 並べ替え

| ボタン | 得られること |
|---|---|
| OrderBy | キーで並べ替える動きを確認できます |
| ThenBy | 2段目以降の並べ替え（同値のときの追加キー）を確認できます |

### 集合演算

| ボタン | 得られること |
|---|---|
| Distinct | 重複を除いて一意にする動きを確認できます |
| Union | 2つを合わせて重複を除く動きを確認できます |
| Intersect | 共通部分だけ残す動きを確認できます |
| Except | 片方にだけある要素を残す動きを確認できます |

### 量の確認

| ボタン | 得られること |
|---|---|
| Any | 条件に合う要素が1つでもあるかを確認できます |
| All | 全要素が条件に合うか（全部がOKか）を確認できます |

### 集計

| ボタン | 得られること |
|---|---|
| Count | 件数の数え方を確認できます |
| Sum | 合計の出し方を確認できます |
| Average | 平均の出し方を確認できます |
| Min | 最小値の取り方を確認できます |
| Max | 最大値の取り方を確認できます |
| Aggregate | 畳み込みで集計する考え方を確認できます |

### 要素取得

| ボタン | 得られること |
|---|---|
| FirstOrDefault | 先頭要素（無ければ既定値）を取る動きを確認できます |
| SingleOrDefault | 要素が1件だけを前提に取る動きを確認できます（複数あると例外になります） |
| ElementAt | 指定インデックスの要素を取る動きを確認できます |
| DefaultIfEmpty | 空シーケンスに既定要素を補う動きを確認できます |

### 結合

| ボタン | 得られること |
|---|---|
| Join | キーで2つのシーケンスを結合する動きを確認できます |
| GroupJoin | 結合結果をグルーピングして受け取る動きを確認できます |

### 変換と生成

| ボタン | 得られること |
|---|---|
| ToList | シーケンスを List に材料化する動きを確認できます |
| ToDictionary | キー指定で Dictionary を作る動きを確認できます |
| ToLookup | 1キーに複数要素をぶら下げる Lookup を作る動きを確認できます |
| Zip | 2つを同じ位置で組にする動きを確認できます |
| Chunk | N 個ずつに分割する動きを確認できます |
| Append | 末尾に1要素を足す動きを確認できます |
| Prepend | 先頭に1要素を足す動きを確認できます |
| Concat | 2つを連結する動きを確認できます |
